# How to run the project
1. Install packages with: <br />
    `yarn install` 
     <br /> or 
     `npm install`

2. Run frontend app and backend (node) app with (backend is MAC OS only):  <br /> 
    `yarn start` 
     <br /> or 
     `npm run start`

    
3. Cool stuff
